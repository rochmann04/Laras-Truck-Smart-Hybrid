Tentang LARAS TRUCK

Inovasi Smart Hybrid dengan Sistem Penghancur dan Pemilah Sampah Terintegrasi

LARAS TRUCK adalah solusi inovatif untuk mengatasi masalah sampah perkotaan dengan mengintegrasikan teknologi penghancur dan pemilah sampah dalam satu sistem yang efisien. Sistem ini dirancang untuk memproses berbagai jenis sampah (organik dan anorganik) secara otomatis, mengurangi volume sampah hingga 70%, dan memfasilitasi daur ulang yang lebih efektif.

Fitur Utama

  • Sistem pemilahan otomatis dan sensor canggih
  • Penghancur sampah berkapasitas tinggi dengan konsumsi energi rendah
  • Monitoring real-time melalui dashboard terintegrasi
  • Sistem hybrid yang dapat beroperasi dengan listrik atau tenaga surya
  • Pengumpulan data analitik untuk optimasi proses

Pemantauan Berbasis Web

Seluruh proses pada LARAS TRUCK dapat dipantau melalui sistem pemantauan berbasis web yang terhubung langsung dengan Raspberry Pi Model B+. Data dari sensor dikirimkan ke server menggunakan protokol komunikasi seperti MQTT atau HTTP sehingga informasi mengenai jenis sampah, suhu ruang penghancur, kapasitas wadah, serta status motor dapat diperbarui secara real-time. Antarmuka web menampilkan data tersebut dalam bentuk grafik dan indikator status sehingga operator dapat memantau kinerja kendaraan dari jarak jauh. Sistem ini juga mampu memberikan notifikasi ketika terjadi kondisi tertentu seperti beban berlebih, kenaikan suhu yang tidak wajar, atau hambatan pada sistem pemilahan. Dengan demikian, pemantauan berbasis web meningkatkan keandalan operasional LARAS TRUCK serta memastikan proses pengolahan berjalan lebih efisien dan terkendali.
